# Spec Homepage — Yombal.sn

## Objectif

Accueillir le visiteur, lui expliquer Yombal en 5 secondes, et l'orienter vers la bonne entrée : tailleur, tissu ou modèle.

## Sections attendues

### 1. Header / Navigation
- Logo Yombal.sn
- Menu : Tailleurs | Tissus | Modèles | Comment ça marche | Connexion
- Sur mobile : menu hamburger + barre de recherche accessible

### 2. Hero (section principale)
- Accroche courte et forte (ex. "Votre tenue sur mesure, livrée chez vous")
- Sous-titre court (ex. "Choisissez votre tissu, votre modèle et votre tailleur")
- 3 CTA principaux : [Trouver un tailleur] [Choisir un tissu] [Voir les modèles]
- Image hero : belle tenue sénégalaise, ambiance moderne
- Mobile : CTA empilés verticalement, toujours visibles

### 3. Comment ça marche (3 ou 4 étapes)
- Étape 1 : Choisissez votre tissu
- Étape 2 : Sélectionnez un modèle
- Étape 3 : Choisissez un tailleur
- Étape 4 : Commandez et payez en ligne
- Format : icônes simples + texte court + flèche entre étapes

### 4. Tailleurs en vedette
- Grille de 4 à 6 cartes tailleurs
- Chaque carte : photo, nom, ville, spécialité, note, CTA [Voir le profil]
- CTA section : [Voir tous les tailleurs]

### 5. Tissus populaires
- Grille de 4 à 8 cartes tissu
- Chaque carte : photo, nom tissu, prix FCFA, CTA [Voir le tissu]
- CTA section : [Voir tous les tissus]

### 6. Modèles tendance
- Grille de 4 à 8 cartes modèles
- Chaque carte : photo modèle, style, CTA [Choisir ce modèle]
- CTA section : [Voir tous les modèles]

### 7. Bloc confiance
- 3 à 4 arguments : Tailleurs vérifiés | Paiement sécurisé | Suivi de commande | Livraison
- Icônes simples + texte court

### 8. Témoignages clients (optionnel phase 1)
- 2 à 3 avis clients
- Note, prénom, ville, texte court

### 9. CTA bas de page
- "Commencez maintenant" ou "Trouvez votre tailleur"
- Bouton bien visible sur fond de couleur

### 10. Footer
- Liens utiles : À propos | CGU | CGV | Contact | Devenir partenaire
- Réseaux sociaux
- Mention PayTech sécurisé

## CTA principaux

| CTA | Destination |
|-----|-------------|
| Trouver un tailleur | /tailleurs |
| Choisir un tissu | /tissus |
| Voir les modèles | /modeles |
| Devenir partenaire | /partenaire |

## Données affichées

- Tailleurs : photo, nom, ville, spécialité, note
- Tissus : photo, nom, prix FCFA
- Modèles : photo, style

## Contraintes mobile

- Hero : texte court, CTA empilés, image en fond ou en dessous
- Grilles : 2 colonnes max sur mobile
- Navigation : hamburger, pas de menu horizontal
- Boutons : hauteur min 48px pour le tactile

## Points à éviter

- Trop de texte dans le hero
- Menu trop chargé
- Grilles à 4+ colonnes sur mobile
- CTA cachés sous la ligne de flottaison
- Mélange d'informations tissu et confection sans séparation claire
